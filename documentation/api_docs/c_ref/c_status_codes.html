Status Codes#

enum qpl_status#

Intel QPL return status list.

Values:

enumerator QPL_STS_OK#

Processing error base Calculates Processing error base Operation completed successfully

enumerator QPL_STS_BEING_PROCESSED#

Processing error base Calculates Processing error base Job is still being processed

enumerator QPL_STS_MORE_OUTPUT_NEEDED#

Processing error base Calculates Processing error base Decompression operation filled output buffer before finishing input

Todo:

deprecate in the future

enumerator QPL_STS_MORE_INPUT_NEEDED#

Processing error base Calculates Processing error base Compress/Decompress operation need more input

Todo:

deprecate in the future

enumerator QPL_STS_JOB_NOT_CONTINUABLE_ERR#

Processing error base Calculates Processing error base A job after a LAST job was not marked as FIRST

enumerator QPL_STS_QUEUES_ARE_BUSY_ERR#

enumerator QPL_STS_NULL_PTR_ERR#

enumerator QPL_STS_OPERATION_ERR#

enumerator QPL_STS_NOT_SUPPORTED_MODE_ERR#
